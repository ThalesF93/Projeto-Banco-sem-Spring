# Banco sem Spring

Sistema bancário em Java puro, sem frameworks.

## Sobre
Simula operações bancárias como depósito, saque, transferência e geração de extrato.

## Tecnologias
- Java 21
- Maven

## Conceitos aplicados
- Herança e abstração
- Exceções customizadas
- Encapsulamento
- Streams e Lambdas
